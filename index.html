About me

Hello! I'm Spencer Thompson

Web-developer based in Toronto, Canada

I began my journey in Mechanical Engineering at McGill, with a minor in Theatre. After graduation I worked in the fascinating world of machine vision system sales, where I was problem-solving on the fly every day. Then I began to fall in love with acting and improv, culminating in co-founding the improv company The Assembly. My main muse these days is of course code! In my spare time I can be found playing online chess, and cycling throughout the city and beyond!

Portfolio

Not So Trivial Pursuit

A React app that uses a trivia API to generate a series of questions based on the user's category, difficulty and number of questions chosen. Games can be saved (stored in firebase). Created with classmates Nicole Wu, Emily Petracco and Seunggi Lee

Joke Book

A React app that takes a user's text input (a joke) and stores it in firebase. Also allows for each joke to have a star rating that can be customized.

Chess Profiler

A javascript app that uses the chess.com API to display a player's stats. It displays profile info, ratings as well as a more in depth look at recent game data (how many wins by resignation, checkmate etc)

Meme Maker

A javascript app which displays popular memes and allows the user add their own captions, or add random pre-written captions. Made with classmate Sid Lee.
